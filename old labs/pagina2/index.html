¿Qué es Material design?

(Lo voy a poner aquí para que no cambie la estética de la página)

Material Design es un sistema/lenguaje de diseño creado por Google en 2014. Busca ofrecer una guía para interfaces claras, organizadas y fáciles de entender, mejorando la experiencia del usuario mediante principios como espacio, colores y movimientos.

¿Por qué usar JavaScript para validar inputs? Validar datos con JavaScript en el cliente es útil porque ayuda al usuario a corregir errores de forma inmediata, como un correo inválido o campos vacíos, sin tener que esperar la respuesta del servidor. También mejora la eficiencia al evitar que datos incorrectos lleguen al backend, reduciendo la carga del servidor.

¿Cómo se pueden burlar estas validaciones? Las validaciones hechas con JavaScript pueden ser manipuladas porque el usuario tiene control sobre el navegador. Por ejemplo, pueden desactivar JavaScript, editar el código usando DevTools o enviar datos directamente al servidor con herramientas como Postman.

¿Por qué sigue siendo buena práctica? Aunque no son seguras, las validaciones en el cliente mejoran la experiencia y optimizan recursos. Sin embargo, siempre deben complementarse con validaciones en el servidor, que son esenciales para garantizar la seguridad de los datos.
